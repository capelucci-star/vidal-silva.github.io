Os 5 Pilares do Meu Prompt Base

O framework que uso para extrair o máximo da IA em qualquer contexto — do LinkedIn ao planejamento estratégico.

Prompt Base — Vidal Silva 2026
1. Definição da Tarefa com Critério de Sucesso
Não peça apenas um resumo. Peça: "Resuma este artigo técnico para que eu possa usá-lo como argumento em uma reunião de diretoria com foco em ROI." Se a IA sabe o critério de sucesso (convencer a diretoria), o tom muda completamente.
2. Contexto Real (Grounding)
Forneça os documentos. Em vez de perguntar sobre o mercado de educação, suba o Censo Escolar 2025. A IA trabalha sobre fatos, não sobre o que ela "acha". Este passo é o favorito de quem usa NotebookLM.
3. Decomposição de Referência
Se você gosta do estilo de escrita do Murillo Leal, não diga apenas "escreva como ele". Peça para a IA: "Analise este texto, identifique o uso de metáforas, a estrutura de frases curtas e o tom provocativo. Agora, aplique esse padrão ao meu tema."
4. Briefing de Saída
Defina o formato da entrega. Você quer um parágrafo para o LinkedIn? Um bullet point para o blog? Um script de áudio para o Notebook? Definir o output economiza 90% do seu tempo de edição.
5. Acordo de Restrições (Constraints)
Este é o passo da Maturidade. Diga o que você NÃO quer. Ex: "Não use jargões de marketing, não cite Python ou MBA de forma direta, e não exceda 100 palavras." Isso elimina o ruído e garante que a entrega seja exatamente o que você precisa.
